# NFE Platform Prototype

This is a browser-based starter prototype for the NFE Platform.

## Included Areas

- Public Website
  - What is NFE?
  - Documentation
  - Benchmark Results
  - Blog
  - Downloads

- NFE Studio
  - Analysis Workspace
  - Layer Visualization
  - Case History
  - Evidence Ledger
  - Report Export

- NFE-OS
  - Project Management
  - Artifact Archive
  - Version Control
  - Governance

- Administration
  - Users
  - Roles
  - Metrics
  - System Settings

## How to Run

Open `index.html` in a browser.

No server or installation is required for this prototype.

## Next Build Step

Convert this static prototype into a full app using:

- Frontend: Next.js or React
- Backend: FastAPI or Node/Express
- Database: PostgreSQL or SQLite for local development
- Authentication: owner/admin/collaborator/viewer roles
- Exports: JSON, TXT, PDF, DOCX
